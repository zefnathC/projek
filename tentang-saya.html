FELISIA MARIA WAAS MASCARENHAS

Tentang Saya



Nama saya Felisia Maria Waas Mascarenhas biasa dipanggil Fel. Saya lahir di Kupang, 09 Desember 2002, saya anak pertama dari empat bersaudara, ayah saya bernama Felisberto Mascarenhas dan ibu saya bernama Johanna Ferderika Waas. Ayah bekerja sebagai Pegawai Negeri Sipil dan ibu saya sebagai Ibu Rumah Tangga.
Saya bersekolah SD di SD GMIT Oepura, saya melanjutkan SMP di SMP Negeri 1 Kota Kupang, yang letaknya kurang lebih 3 kilo dengan sekolah SD saya. Saya melanjutkan SMK di SMK Negeri 1 Kota Kupang, disitulah saya mengenakan seragam putih abu-abu dan menuntut ilmu selama 3 tahun.
Saya mengambil jurusan Teknik Komputer dan Jaringan (TKJ) saat SMK, dan melanjutkan pendidikan di Universitas Teknologi Digital Indonesia dengan Fakultas Teknologi Informasi jurusan Informatika di kota Yogyakarta.
Semenjak duduk dibangku SMK saya pernah mengikuti osis dalam periode 2017-2018. Selesai kepengurusan saya mengikuti ekstrakulikuler basket, tim teknisi dan juga menjadi kandidat siswa dalam lomba olimpiade matematika tingkat kota dan provinsi. Singkat cerita di 2019 saya magang ke Malang dan setelah selesai magang saya mengikuti sertifikasi profesi (LSP) skala level 2.
Memasuki bangku perkuliahan saya mengikuti beberapa organisasi diantaranya IKPM Ambon-Yogyakarta periode 2021-2022, disana saya menduduki jabatan sebagai sekretaris divisi kajian strategi, didalam lingkup kampus saya mengikuti beberapa organisasi diantaranya KMK UTDI dan BEM UTDI. Di BEM sendiri saya menduduki jabatan sebagai menteri luar negeri
